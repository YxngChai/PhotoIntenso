# PhotoIntenso

Exercice for Kercode school.
Photo Intenso

A landing page recreated from a provided design mockup using HTML5 and CSS3.

## Features

- Semantic HTML structure for better accessibility and SEO
- Flexbox-based layout
- Custom typography using Google Fonts
- CSS variables for colors and fonts
- Hover effects and interactive links
- Accessible images with descriptive alt attributes
- Clean BEM-inspired CSS organization
